Apple Music Playlist Importer

Info: Since Apple Music's AppleScript API is limited, use the links below to search for each song in Apple Music on your device or web.
How to add songs:
  1. Upload a Spotify playlist JSON export file using the button below
  2. Click "Process Songs" to extract and load the playlist
  3. Click the "Search" link for each song
  4. Apple Music will open and search for the song
  5. Click the "+" button to add it to your library or a playlist
  6. Or copy the song/artist names and search manually
